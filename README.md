# 🏠 Nestly

Nestly is a smart real estate listing app that helps users find the best rental options using a custom scoring system and interactive map visualization.

---

## 🚀 Features

- 📍 Interactive map (Leaflet + OpenStreetMap)
- 🏆 AI-style listing scoring system
- 💰 Price filtering
- 🛏 Bedroom filtering
- 📊 Ranked listings (best matches first)
- 🗺 Synchronized listings + map markers

---

## 🧠 How It Works

Each listing is scored based on:

- Price range
- Number of bedrooms
- Property type (condo bonus)
- Distance to subway
- Distance to grocery stores
- Parking availability

Higher score = better recommendation.
